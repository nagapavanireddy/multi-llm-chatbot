# multi-llm-chatbot
Multi-LLM Chatbot (Basic Version)
📌 Overview

This project is a simple command-line chatbot that simulates responses from multiple AI models such as LLaMA and Qwen.
It is designed as a beginner-friendly implementation to understand how multi-model AI systems work.

🚀 Features
Accepts user input from terminal
Simulates responses from:
LLaMA (Meta AI model)
Qwen (Alibaba AI model)
Simple and clean command-line interface
Easy to extend with real AI APIs in future
🛠️ Tech Stack
Python
Command Line Interface (CLI)
📂 Project Structure
multi-llm-chatbot/
│── app.py
│── README.md
│── requirements.txt (optional)
▶️ How to Run
Open project in VS Code
Run the following command:
python app.py
Enter any question in terminal
💡 Example

Input:

hi   what is AI?

Output:

LLaMA Response:
LLaMA says: AI is intelligence shown by machines.

Qwen Response:
Qwen says: AI helps automate tasks and learning.
